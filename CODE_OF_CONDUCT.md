# Code of Conduct

## Our Pledge
We pledge to create a welcoming and inclusive environment.

## Our Standards
Be respectful, inclusive, and constructive in all interactions.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing this code.

## Scope
This code applies to all project spaces and communication channels.

## Enforcement
Incidents may be reported to the maintainers for review.

## Enforcement Guidelines
Maintain transparency and fairness while handling reports.

## Attribution
Based on the Contributor Covenant Code of Conduct.
